# Lab: Basic Connectivity (SOHO Simulation)

## 📌 Overview
This project demonstrates **Module 1: Communication in a Connected World**. I built a Small Office/Home Office (SOHO) network to visualize how data transmission works across a LAN and through a Gateway.

## 🛠️ Network Components
* **2 PCs:** End-user devices.
* **1 Switch (2960):** Handles local traffic using MAC addresses.
* **1 Router (4331):** The Default Gateway connecting the LAN to other networks.

## 🧠 Key Logic Learned
1.  **Encapsulation:** Watching the ICMP packet get "wrapped" in headers as it moves through the OSI layers.
2.  **The Router's Role:** While the switch handles communication between PC1 and PC2, the Router is essential for any traffic intended for a different network.
3.  **Troubleshooting "Red X":** I learned to filter traffic in Simulation Mode to distinguish between background protocols (STP/DTP) and my actual data (ICMP).
